# CyanPrint Searching Skill

Use this before adding a dependency.

- Search templates: `cyanprint search --kind template <query>`.
- Search processors: `cyanprint search --kind processor <query>`.
- Search plugins: `cyanprint search --kind plugin <query>`.
- Search resolvers: `cyanprint search --kind resolver <query>`.
- Prefer stable, declared registry refs over copying behavior into this artifact.
