# CyanPrint New Artifact Template

Creates a CyanPrint v4 artifact scaffold for templates, processors, plugins, and resolvers.

The generated project includes:

- folder-first artifact files
- `cyan.test.yaml` fixtures
- GitHub CI that installs `cyanprint` from the Gemfury apt repository on Ubuntu
- LLM-friendly skills for creating, updating, pushing, documenting, and searching artifacts
